## Algorithm
**TF-IDF Vectorisation + Cosine Similarity**

- Each movie is converted into a TF-IDF vector using its Description,
  Genres, Directed by, and Written by fields.
- Recommendations are generated by computing cosine similarity between
  the query movie's vector and every other movie's vector.
- The top-5 most similar movies are returned, ranked by score.

No external ML framework is required — the entire engine is built using
pure Python (math, re, collections).

## Input
A single movie title string, exactly as typed by the user in the web form.

```python
inputs = {'title': 'The Godfather'}
```

The function handles:
- Exact title matching (case-insensitive)
- Partial title matching via search fallback
- Empty / invalid input → returns readable error message, never crashes
